# ConfusedDIS

混淆字典表

难以辨认的中文字符

混淆后难以辨认的代码

堪称泥石流

## 如何使用 
在proguard-rules.pro文件中进行配置
-obfuscationdictionary ConfusedDIS.txt
-classobfuscationdictionary ConfusedDIS.txt
-packageobfuscationdictionary ConfusedDIS.txt

ConfusedDIS 文件下提供了大量的词典 可以根据需要自行删减
